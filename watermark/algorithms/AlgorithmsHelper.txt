Тут хранится подсказка по данному блоку. 
В данной папке watermark_project\watermark\algorithms хранятся основные алгоритмы и "ядро" программы.

1. Интерфейс каждого алгоритма (например, lsb.py / dct.py / dwt.py / cnn_autoencoder.py):
В каждом из этих файлов должны быть реализованы две основные функции (или классы с этими методами):

embed(image: np.ndarray, secret: Any, params: dict) -> np.ndarray

“Внедрить” водяной знак в изображение.

image — исходное изображение как numpy-массив.

secret — водяной знак (строка/байты/картинка).

params — словарь параметров, влияющих на внедрение (опции).

Возвращает: новое изображение с встраиванием.

extract(image: np.ndarray, params: dict) -> Any

“Извлечь” водяной знак из изображения.

image — изображение с водяным знаком.

params — параметры извлечения.

Возвращает: извлечённый секрет.
